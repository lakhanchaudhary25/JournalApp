# Journal App (Spring Boot)

A RESTful backend application built using Spring Boot to manage journal entries.

## Features

* Create, Read, Update, Delete (CRUD) journal entries
* REST API architecture
* Structured layered design (Controller → Service → Repository)

## Tech Stack

* Java
* Spring Boot
* Maven

## Upcoming Features

* Authentication & Authorization (Spring Security)
* Exception Handling

## How to Run

```bash
./mvnw spring-boot:run
```
